## curiosities
**Windows 95 Installation Medium** : Before the widespread adoption of CD-ROMs for software distribution, installing large programs like Windows 95 was a cumbersome process reliant on floppy disks. Windows 95, being nearly 60 megabytes, required 26 individual 1.44 megabyte floppy disks for installation, a lengthy process prone to errors, which highlighted the need for more efficient storage and installation methods.
