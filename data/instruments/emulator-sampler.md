# instrument - Emulator Sampler

## curiosities
**Sampling Kraftwerk for "Blue Monday"** : A choral effect from Kraftwerk's song "Uranium" was sampled and incorporated into New Order's "Blue Monday" using the Emulator sampler. "They sampled a choral effect from a craftwork song called Uranium..." ← https://www.youtube.com/watch?v=dablAKDOOV0
**Used in "Blue Monday"** : New Order used an Emulator sampler, connected to a sequencer built by Stephen Morris, as a crucial component in the intricate electronic setup for their song "Blue Monday." "He built a sequencer that could be connected to the band's new sampler, a keyboard called an emulator." ← https://www.youtube.com/watch?v=dablAKDOOV0
