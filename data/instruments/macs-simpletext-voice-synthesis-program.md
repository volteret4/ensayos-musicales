# instrument - Mac's SimpleText Voice Synthesis Program

## curiosities
**"Fitter Happier" Vocal Performance** : The speech function from an old Mac's SimpleText voice synthesis program was used to generate the lead vocals for Radiohead's 1997 song "Fitter Happier." Thom Yorke, experiencing writer's block, fed slogans into the program to create the "unsettling and powerful" spoken word track. "In this case, it's the speech function from simple text an old Mac voice synthesis program" ← https://www.youtube.com/watch?v=M1JTdUxOEMU
