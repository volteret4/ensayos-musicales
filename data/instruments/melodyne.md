# instrument - Melodyne

## curiosities
**Transparent Vocal Tuning Program** : Melodyne is described as the "most transparent tuning program" Chris Berkett has discovered, offering a superior alternative to Auto-Tune. He uses it selectively to correct only specific problematic notes in a vocal performance, preserving the natural feel of the rest of the take. "I use a program called Melodyne, which is the most transparent tuning program I've ever discovered." ← https://www.youtube.com/watch?v=mjQBaX_3oL4
