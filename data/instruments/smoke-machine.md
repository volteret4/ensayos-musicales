# instrument - Smoke Machine

## curiosities
**First Stage Prop on Warped Tour** : The smoke machine was the very first stage prop to be used on the Warped Tour, introduced by the band Avenged Sevenfold. "The first band to use any kind of prop on stage was a Venge 7 Fold. And the prop they brought with them? Smoke Machine." ← https://www.youtube.com/watch?v=7jfQaUV6Bq4
