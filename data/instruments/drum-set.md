## curiosities

**Larry Mullen's Secondhand Kit** : Larry Mullen Jr.'s mother spent 17 pounds on a secondhand drum set for him. ← https://www.youtube.com/watch?v=gsSTJWN6cUo
**Tommy Ramone's Budget Kit** : Tommy Ramone, the drummer for The Ramones, was "forced to keep time" using a $100 drum set. ← https://www.youtube.com/watch?v=gsSTJWN6cUo
