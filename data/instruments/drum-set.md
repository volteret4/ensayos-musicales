# instrument - Drum set

## curiosities
**Joey Ramone's Expanded Kit** : Joey Ramone's grandmother gave him a full drum set at age 13, which he expanded to be as large as Keith Moon's.
**Larry Mullen's Secondhand Kit** : Larry Mullen Jr.'s mother spent 17 pounds on a secondhand drum set for him.
**Tommy Ramone's Budget Kit** : Tommy Ramone, the drummer for The Ramones, was "forced to keep time" using a $100 drum set.
