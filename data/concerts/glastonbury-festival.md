# concert - Glastonbury Festival

## curiosities
**Radiohead's Legendary Set (June 28, 1997)** : On June 28, 1997, Radiohead performed a set at Glastonbury Festival that some maintain was "the best show of all time ever." Despite initial technical problems with stage monitors and a spotlight blinding Thom Yorke, the band overcame these issues, and Yorke's request to light the crowd created a moment where "the magic set in," solidifying the band's legend following the release of their album *OK Computer*.
