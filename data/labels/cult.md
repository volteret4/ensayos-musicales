# label - Cult

## curiosities
**Julian Casablancas' Imprint** : Cult is Julian Casablancas' own record label imprint. The Strokes' four-track EP, "Future Present Past," released in 2016, marked the band's first release on this label. "And it was the first strokes release on a label called Cult, which is Julian's imprint." ← https://www.youtube.com/watch?v=jEfofyobGiI
