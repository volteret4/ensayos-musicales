# label - Microsoft

## curiosities
**File Name Suffix Convention** : The file system of Windows 95 mandated that each file be identified by a three-letter suffix. This convention directly led to the abbreviation of "MPIG 3" to ".mp3" for the ISO-Moving Pictures Expert Group Layer 3 format, standardizing the widely recognized file extension. "Windows 95's file system required that each file be identified by a three-letter name suffix at the end." ← https://www.youtube.com/watch?v=v7DXXu_1fvM
**Windows 95 Operating System** : Microsoft's release of the "groundbreaking" Windows 95 operating system necessitated the development of compatible MP3 encoding software. Bernhard Grill was tasked with creating a Layer 3 encoder for Windows 95, which resulted in the creation of Winplay 3, a program small enough to fit on a 1.4-megabyte floppy disk. "Make a Lair 3 encoder for Microsoft. They were about to release a groundbreaking operating system called Windows 95." ← https://www.youtube.com/watch?v=v7DXXu_1fvM
**Windows 95 and CD-ROM Adoption** : Microsoft's release of Windows 95 on August 15, 1995, was a pivotal moment that cemented the company's dominance in personal computing. The operating system's large size (nearly 60 MB, requiring 26 floppy disks for installation) prompted Bill Gates to include CD-ROMs as the installation medium. This decision led to PCs widely shipping with CD-ROM drives, inadvertently laying the groundwork for digital music ripping and file sharing by making CD-ROM drives ubiquitous and enabling users to treat music CDs as data disks. "Then on August 15, 1995, Microsoft introduced Windows 95." ← https://www.youtube.com/watch?v=NAroxFbYkRw
