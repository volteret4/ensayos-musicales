"I've used it on Steve Earle, Chanel, O'Connor, Buffy St. Marie, Alison Moyer, they've all sung in that, that mic, it sounds great." ← https://www.youtube.com/watch?v=mjQBaX_3oL4
**Work on "Copperhead Road"** : Chris Berkett did some work on Steve Earle's "Copperhead Road." "Steve, all I did some work on Copperhead Road." ← https://www.youtube.com/watch?v=mjQBaX_3oL4
