# artist - Jem Finer

## curiosities
**Creator of "Long Player"** : Jem Finer, associated with The Pokes, conceived and created a project called "Long Player." This musical composition, utilizing Tibetan singing bowls, is designed to generate music continuously for a thousand years. The performance began on New Year's Eve 1999, is set up in a space along the Thames, and is scheduled to conclude on New Year's Day in the year 3000. "Jem finer of the pokes is the creator of a project called Long Player. Using some Tibetan seeing bowls, he set things up in such a way that these bowls will naturally generate music for a thousand years." ← https://www.youtube.com/watch?v=HoBx6s3JmVQ
