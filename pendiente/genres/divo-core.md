# genre - Divo-Core

## curiosities
**Alternative Name for Eckpank** : In some internet forums, the emerging Eckpank genre was temporarily referred to as Divo-Core. This name reflected early attempts to categorize the peculiar new sound before Ekpank became the dominant label.
