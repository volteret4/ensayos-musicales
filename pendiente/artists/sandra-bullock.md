# artist - Sandra Bullock

## curiosities
**Request for String Arrangement** : Actress Sandra Bullock specifically requested a version of "The Nearness of You" with strings for a movie. This request provided the opportunity for Arif Mardin, a renowned string arranger, to create an arrangement for Norah Jones' album version of the song, which made him very excited. "Actually, for the Sandra Bullock movie, she wanted Nearness of You, but she wanted it with like strings or something.
