# artist - Castello Varsky

## curiosities
**Ultimate Soundtracker Creator (1967)** : Castello Varsky created the Ultimate Soundtracker in 1967 for the Commodore Amiga, which was the first tracker software and only allowed four simultaneous audio channels. He also created the .mod format, which became the standard file format for piracy. The tracker did not export audio files like MP3s, but instead saved instructions and a "little box with loose samples" in a .mod file, which was then executed in real-time by the device. "El primer track fue el Ultimate Soundtracker, creado por Castelo Varsky en 1967 para el comodore amiga, que solo permitía cuatro canales de audios simultáneos.
