# artist - Bruce Lundvall

## curiosities
**Overseeing Manhattan Records** : Bruce Lundvall oversaw Manhattan Records, a sister label to Blue Note, which Arif Mardin either started or revamped. This label focused on classical, musical theater, and adult contemporary genres.
