## curiosities
**3 1/2-inch Floppy Disk Adoption** : The Amiga adopted the 3 1/2-inch floppy disk as its standard storage medium, following its standardization by the Atari ST. This format became prevalent in personal computing. **Ultimate Soundtracker Platform** : The Commodore Amiga was the platform for which Castello Varsky created the Ultimate Soundtracker in 1967, which was the first tracker software. This software allowed for four simultaneous audio channels.
