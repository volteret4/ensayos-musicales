## curiosities
**First Tracker Software (1967)** : The Ultimate Soundtracker, created by Castello Varsky in 1967 for the Commodore Amiga, was the first tracker software. It allowed for only four simultaneous audio channels and pioneered the .mod file format, which became standard for digital piracy.
