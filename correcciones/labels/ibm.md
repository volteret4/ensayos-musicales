# label - IBM

## curiosities
**Early Operating System Rivalry (OS/2)** : IBM's OS/2 operating system was a competitor to Microsoft Windows 3.1 in the early 1990s, engaged in a "real dogfight" to develop an easier graphical user interface for personal computers. This competition, while not ultimately leading to OS/2's dominance, was part of the technological landscape that pushed for more user-friendly computing, which indirectly facilitated future music technology developments.
