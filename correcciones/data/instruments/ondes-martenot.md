# instrument - Ondes Martenot

## curiosities
**Electronic Keyboard History** : The Ondes Martenot is an "old electronic keyboard invented back in the 1920s." Jonny Greenwood played this instrument on Radiohead's "Kid A" album. It produces an "eerie" sound often compared to a woman's voice, famously used in the original theme for "Star Trek."
