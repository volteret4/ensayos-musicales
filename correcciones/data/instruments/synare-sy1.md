# instrument - Synare S.Y.1

## curiosities
**Real-time Drum Synthesizer for "Blue Monday"** : What sounds like a brittle snare drum in New Order's "Blue Monday" was actually Stephen Morris physically hitting pads connected to a Synare S.Y.1 unit, which functioned as a real-time drum synthesizer. "that's actually Stephen Morris physically hitting pads connected to an S.Y.1 Sincasher unit. So you can call that a real-time drum synthesizer." ← https://www.youtube.com/watch?v=dablAKDOOV0
